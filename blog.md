## 学习一些Markdown语法
发现hexo搭站之后，使用的文章语法是markdown，这和gitbook的语法是一样一样的，之前就没有学，既然如此，不如迅速的学习一下
### 首先，理所当然的是标题的用法
#### 方法一 #
'#'在markdown里是区分标题的一种方式，共有6级，以文字前#的多少作为区分
```
#  一个为一级标题
## 两个为二级标题
 ** 以此类推到6级标题
```
这篇文章的多级标题是不是用的很好啊？其实也是遇到了一点点问题的，注意标记标题的#号和标题文字之间需要空格
#### 方法二 =和-
= 和 - 标记语法格式如下：
```
我展示的是一级标题
=================
我展示的是二级标题
-----------------
```
